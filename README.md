# Power BI Data Modeling Project

## Project Overview

This project demonstrates the design and implementation of a dimensional data model in Power BI using Star Schema principles. The objective was to create a scalable and efficient data model that supports business reporting and analytics across multiple business functions.

## Data Model Structure

### Fact Tables

* Fact Sales
* Fact Inventory
* Fact Campaign Spend
* Fact Promotion Coverage
* Fact Order Process
* Fact Sales Target

### Dimension Tables

* Dim Customer
* Dim Product
* Dim Geography
* Dim Date
* Dim Campaign
* Dim Order Flags

## Data Modeling Concepts Applied

* Star Schema Design
* Fact and Dimension Modeling
* One-to-Many Relationships
* Data Normalization for Analytics
* Relationship Management
* Business-Oriented Data Architecture

## Business Use Cases

The model supports analysis of:

* Sales Performance
* Inventory Monitoring
* Marketing Campaign Effectiveness
* Promotion Coverage Tracking
* Order Processing Analysis
* Sales Target Achievement

## Tools Used

* Power BI Desktop
* Power Query
* Data Modeling

## Key Learning Outcomes

* Designing scalable Power BI data models
* Implementing Star Schema architecture
* Managing relationships between multiple fact and dimension tables
* Building analytics-ready datasets for business intelligence reporting
